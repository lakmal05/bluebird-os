# Bluebird OS

**Bluebird OS** is a lightweight experimental operating system built from scratch.  
The project is focused on learning low-level system design, kernel development, memory management, process scheduling, file systems, and hardware interaction.

## Overview

Bluebird OS is created as a personal operating system development project.  
The main goal is to understand how an operating system works from the ground up, starting from the boot process and gradually building core OS features.

This project is mainly for learning, research, and experimentation.

## Features

- Custom bootloader support
- Basic kernel development
- Text output to screen
- Keyboard input handling
- Memory management experiments
- Process scheduling concepts
- File system research
- Hardware-level programming practice

## Project Goals

The main goals of Bluebird OS are:

- Learn how computers boot into an operating system
- Understand kernel architecture
- Explore memory management
- Build simple hardware drivers
- Study process and task management
- Create a small but expandable operating system
- Improve low-level programming skills

## Tech Stack

- C
- Assembly
- GCC cross-compiler
- Makefile
- QEMU for testing
- Git and GitHub for version control
